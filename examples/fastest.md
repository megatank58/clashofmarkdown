**Goal**

> You need to print the factorial of a number.
> For example if input is equal to 4, then output should be 24

`Input`
```
Line 1: A Positive integer n
```

`Output`
```
Factorial of the number
```

`Constraints`
```
1 ≤ n ≤ 10
```

**Example**
> __Input:__
> ```
> 4
> ```
> __Output:__
> ```
> 24
> ```
